Question-1: What are the different ways to select an element in the DOM?





Dom(Documents object model) এ এলিমেন্ট সিলেক্ট করার অনেক উপায় আছে যেমন getElementById = ID দিয়ে সিলেক্ট

getElementsByClassName = Class দিয়ে সিলেক্ট

getElementsByTagName = Tag name দিয়ে সিলেক্ট

querySelector = CSS সিলেক্টর (প্রথম ম্যাচ) querySelectorAll = CSS সিলেক্টর (সব ম্যাচ)

Question-2: What is the difference between innerHTML, innerText, and textContent ?





innerHTML: একটি এলিমেন্টের HTML কন্টেন্ট, যার মধ্যে ট্যাগও অন্তর্ভুক্ত থাকে।

innerText: দৃশ্যমান টেক্সট কন্টেন্ট ফেরত দেয়, HTML ট্যাগ বাদ দিয়ে এবং CSS স্টাইল অনুসরণ করে।

textContent: এলিমেন্টের সব টেক্সট কন্টেন্ট ফেরত দেয়, গোপন টেক্সটসহ, স্টাইল বা দৃশ্যমানতার পার্থক্য না করে।

Question-3: What is event delegation in the DOM?





ইভেন্ট ডেলিগেশন হলো DOM এ একটি কৌশল যেখানে প্রতিটি উপাদানে আলাদা করে ইভেন্ট লিসেনার সংযুক্ত করার পরিবর্তে একটি প্যারেন্ট উপাদানে একটি ইভেন্ট লিসেনার সংযুক্ত করে। ইভেন্টটি লক্ষ্য উপাদান থেকে প্যারেন্ট উপাদানে "বাবল" হয়, এবং প্যারেন্ট ইভেন্টটি ম্যানেজ করে।

Question-4: What is event bubbling in the DOM?





ইভেন্ট বাবলিং DOM এ একটি প্রক্রিয়া যেখানে কোনো ইভেন্ট, যেমন ক্লিক, একটি উপাদানে ঘটলে তা প্রথমে সেই উপাদানে ঘটতে থাকে, তারপর "বাবল" হয়ে উপরের প্যারেন্ট উপাদানে পৌঁছায়, এবং এই প্রক্রিয়া ডকুমেন্টের রুট (root) পর্যন্ত চলতে থাকে। যখন কোনো ইভেন্ট চাইল্ড উপাদানে ঘটবে, তখন প্রথমে ওই চাইল্ড উপাদানে ইভেন্ট লিসেনার চলবে। তারপর সেই ইভেন্টটি প্যারেন্ট উপাদানে বাবল হয়ে চলে যাবে, যদি প্যারেন্টে সেই ইভেন্টের জন্য কোনো লিসেনার থাকে, তাহলে সেটি ট্রিগার হবে। এই প্রক্রিয়া চলতে চলতে ইভেন্টটি document পর্যন্ত পৌঁছায়
